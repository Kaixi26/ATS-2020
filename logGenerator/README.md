# logGenerator
## How to run
Install [haskell stack](https://docs.haskellstack.org/en/stable/install_and_upgrade/#installupgrade):
`curl -sSL https://get.haskellstack.org/ | sh`

Build project:
`stack build`

Output usage:
`stack exec -- --help`

Example usage:
`stack run -- --users 4 --orders 10 --accepted 5`
